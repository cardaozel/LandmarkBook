# LandmarkBook

An iOS app showcasing world-famous landmarks. Browse the list and tap to view images and details of each landmark.

## Features

- TableView list of landmarks
- Landmarks: Colosseum, Eiffel Tower, Great Wall of China, Kremlin, Pyramids of Giza, Stonehenge, Machu Picchu
- Detail view with image
- Clean list UI

## Requirements

- Xcode 15+
- iOS 15.0+

## Getting Started

1. Clone and open in Xcode:
   ```bash
   git clone https://github.com/cardaozel/LandmarkBook.git
   cd LandmarkBook
   open LandmarkBook.xcodeproj
   ```

2. Run on simulator or device (⌘R)
3. Tap a landmark to see its image and details

## Project Structure

- `ViewController.swift` — Landmark list (TableView)
- `DetailsVC.swift` — Landmark detail with image
